# 🗓️ DataInsight AI - 14 天高强度开发计划

> **时间投入**：下一周 + 下下周，每天约 7 小时，总计约 14 天
> **核心目标**：完成 DataInsight AI 智能数据洞察平台，可演示、可写入简历

---

## 📌 阶段划分

| 阶段 | 天数 | 核心任务 | 产出物 |
|------|------|---------|--------|
| **阶段一：基础准备** | Day 1-2 | 环境搭建 + Python/FastAPI 学习 | 可运行的后端 Hello World |
| **阶段二：后端骨架** | Day 3-5 | 数据库设计 + 文件上传 + 数据解析 | 可上传文件并查看数据表格 |
| **阶段三：AI 核心** | Day 6-8 | Kimi API 集成 + 智能分析 + 流式输出 | 上传文件后 AI 自动分析并输出 |
| **阶段四：前端开发** | Day 9-11 | React 前端 + UI 组件 + 前后端联调 | 完整可交互的 Web 界面 |
| **阶段五：功能完善** | Day 12-13 | 可视化 + 报告 + 对话追问 | 支持图表、报告导出、对话 |
| **阶段六：收尾优化** | Day 14 | 测试 + 文档 + 部署 + 简历整合 | 可演示的项目 + 简历描述 |

### **Day 7 — AI 数据分析 Prompt 设计**
**目标**：设计 Prompt，让 AI 能够分析数据并结构化输出

| 时间段 | 任务 | 说明 |
|--------|------|------|
| 上午 (3h) | 设计数据分析 Prompt | 角色设定 + 数据描述 + 输出格式要求（JSON） |
| | 测试 Prompt 效果 | 用真实 CSV 数据测试，调整 Prompt |
| 下午 (3h) | 实现 AI 分析服务 | `ai_analyzer.py`：接收 DataFrame，生成分析结果 |
| | 实现分析接口 | `/api/analyze/{id}` 触发分析 |
| 晚上 (1h) | 存储分析结果 | 将 AI 分析结果存入数据库 |

**产出物**：
- [ ] Prompt 能稳定输出结构化分析结果
- [ ] 分析接口能正常工作
- [ ] 分析结果存入数据库

**Prompt 示例思路**：
```
你是一位数据分析师。以下是用户上传的数据文件概览：
- 文件名：{filename}
- 行数：{rows}
- 列数：{cols}
- 列信息：{columns_info}
- 前 5 行数据：{sample_data}

请完成以下分析：
1. 数据整体描述
2. 每列的数据类型和分布特点
3. 可能的异常值或缺失值问题
4. 列之间的关联性分析
5. 数据趋势和洞察

请用中文回答，结构化输出。
```
