# 📊 DataInsight AI - 技术实现方案

> 本方案描述 DataInsight AI 的技术选型与实现路径，按功能模块组织，不暴露具体开发时间线。

---

## 模块一：基础架构搭建

### 目标
建立前后端分离的基础框架，确保项目可运行。

### 后端
- FastAPI 主应用入口（`main.py`）
- 跨域 CORS 配置
- 数据库连接与 ORM 初始化（SQLAlchemy + SQLite）
- 环境配置管理（`pydantic-settings` + `.env`）
- 路由注册与模块划分

### 前端
- Vite + React 18 + TypeScript 项目初始化
- React Router 路由配置（首页 / 详情 / 分析 / 图表 / 报告）
- Axios 请求封装与 API 客户端
- Ant Design 全局引入

### 产出物
- 可运行的前后端骨架
- Swagger API 文档可访问

---

## 模块二：数据库与数据模型

### 目标
设计并创建数据表，支持分析任务、对话记录、图表配置的持久化。

### 数据表
- `analysis_tasks`：任务元数据（文件名、行列数、列信息、AI摘要、状态）
- `chat_messages`：用户与AI的多轮对话记录
- `visualizations`：图表配置（类型、ECharts JSON、描述）

### 关键设计
- 使用 SQLAlchemy ORM 定义模型关系
- 启动时自动建表（`Base.metadata.create_all`）
- 依赖注入获取数据库会话（`get_db`）

### 产出物
- 完整的数据库模型
- 可通过 API 进行 CRUD 操作

---

## 模块三：文件上传与数据解析

### 目标
实现 CSV/Excel 文件的接收、存储与解析。

### 后端
- `multipart/form-data` 文件上传接口
- 文件类型校验（`.csv`, `.xlsx`, `.xls`）
- 文件大小限制（10MB）
- 本地存储（`data/uploads/`）
- Pandas 读取与元数据提取（行数、列数、类型、样本值）

### 前端
- 拖拽上传组件（Ant Design Upload）
- 上传后解析结果展示（列名、类型、样本值）
- 上传成功跳转详情页

### 产出物
- 用户可上传文件并看到解析结果

---

## 模块四：数据预览与统计

### 目标
提供数据表格预览和统计概览。

### 后端
- 分页数据读取接口（支持 `limit`/`offset`）
- 统计概览接口：总行数、总列数、缺失值、重复值
- 列级统计：均值、中位数、标准差、最小值、最大值（数值型）；Top 值分布（分类型）

### 前端
- 数据表格展示（Ant Design Table，支持分页）
- 统计卡片（行数、列数、缺失值、重复行）
- 列统计详情表格

### 产出物
- 数据详情页可查看表格和统计数据

---

## 模块五：AI 智能分析

### 目标
集成大模型 API，实现数据分析的自动化与流式输出。

### 后端
- Moonshot AI (Kimi) API 封装（`httpx` 异步调用）
- Prompt Engineering：角色设定 + 数据描述 + 结构化输出要求
- 数据分析维度：整体描述、质量评估、各列分析、关联分析、业务洞察、可视化建议
- SSE 流式输出（Server-Sent Events）
- 分析结果持久化到数据库

### 前端
- 流式分析结果展示（逐字显示效果）
- "开始分析"按钮触发后端分析
- 使用 `EventSource` 接收流式数据

### 产出物
- 上传文件后触发 AI 分析，流式显示分析结果

---

## 模块六：对话追问

### 目标
支持用户基于数据上下文进行多轮对话。

### 后端
- 对话历史存储与查询（`chat_messages` 表）
- 对话 Prompt 构建（数据摘要 + 历史记录 + 当前问题）
- SSE 流式对话输出
- AI 回复自动存入数据库

### 前端
- 对话式 UI（类似聊天界面，用户消息在右，AI 在左）
- 输入框支持回车发送
- 流式显示 AI 回复

### 产出物
- 用户可在 AI 分析后进行追问（如"哪个品类销售额最高？"）

---

## 模块七：可视化图表

### 目标
根据数据特征自动生成多种图表。

### 后端
- 自动识别列类型（数值型、分类型、日期型）
- 图表推荐策略：
  - 数值列 → 分布柱状图
  - 分类列 → 饼图（Top 8）
  - 两数值列 → 散点图
  - 分类 + 数值 → 分组柱状图
  - 多数值列 → 相关性热力图
- 生成 ECharts 配置 JSON
- 图表配置存入数据库

### 前端
- 图表画廊（Tabs 切换）
- 使用 `echarts-for-react` 渲染图表
- "生成图表"按钮触发后端自动生成

### 产出物
- 自动生成 5-8 张图表，支持切换查看

## 模块八：报告导出

### 目标
生成结构化的 Markdown 数据洞察报告。

### 后端
- 报告模板包含：文件信息、数据概览、列统计表、AI 分析摘要、图表列表、数据样本

- 返回 Markdown 文本内容

### 前端
- 报告预览页面（Markdown 纯文本展示）
- "下载报告"按钮（Blob 下载为 `.md` 文件）

### 产出物
- 一键下载完整的 Markdown 数据洞察报告
